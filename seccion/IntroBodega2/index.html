    Explicación del Proceso ETL - Bloque 2¶

    2.1 - FactAportes¶

    Explicación del Proceso ETL¶

    Este proceso tiene como objetivo gestionar los aportes de diferentes fuentes y preparar los datos para su posterior análisis y carga. Se asegura de estandarizar y transformar la información sobre los aportes para su correcta integración dentro del flujo de datos principal.

    Extracción, Transformación y Carga de Datos¶

    • Extracción: Los datos se extraen de una base de datos relacional utilizando la librería sqlalchemy.
    • Transformación: Se realizan varias transformaciones en los datos usando pandas y numpy, como limpieza de datos nulos y estandarización de formatos de fecha.
    • Carga: Los datos transformados se cargan en la base de datos destino a través de una conexión establecida con sqlalchemy.

    Fragmentos de Código Importantes¶

    import sqlalchemy as sa
from sqlalchemy import create_engine
import pandas as pd
import numpy as np
import time
from dateutil import parser

# Creación del motor de conexión
engine = create_engine('mysql+pymysql://usuario:contraseña@localhost/db_name')

# Extracción de datos
query = 'SELECT * FROM aportes'
df = pd.read_sql(query, engine)

# Transformación de datos
df['fecha'] = df['fecha'].apply(parser.parse)
df.fillna(0, inplace=True)

# Carga de datos
df.to_sql('aportes_limpios', con=engine, if_exists='replace', index=False)

    2.2 - FactEntrega¶

    Explicación del Proceso ETL¶

    Este proceso gestiona la entrega de bienes o servicios y su registro en el sistema, asegurando que la información esté consolidada y lista para el análisis.

    Extracción, Transformación y Carga de Datos¶

    • Extracción: Los datos se obtienen de un sistema de registros utilizando sqlalchemy.
    • Transformación: Se normalizan las columnas, se corrigen errores y se estandarizan los valores de las fechas y montos.
    • Carga: Los datos se almacenan en una tabla preparada para auditorías.

    Fragmentos de Código Importantes¶

    # Extracción de datos
query = 'SELECT * FROM entregas'
df_entregas = pd.read_sql(query, engine)

# Transformación de datos
df_entregas['fecha_entrega'] = pd.to_datetime(df_entregas['fecha_entrega'])
df_entregas['monto'] = df_entregas['monto'].astype(float)

# Carga de datos
df_entregas.to_sql('entregas_auditoria', con=engine, if_exists='replace', index=False)

    2.3 - FactSubsidioVivienda¶

    Explicación del Proceso ETL¶

    Este proceso gestiona la información relacionada con los subsidios de vivienda, asegurando que se registren correctamente y estén listos para análisis de políticas sociales.

    Extracción, Transformación y Carga de Datos¶

    • Extracción: Los datos de subsidios se obtienen de diversas fuentes de registro.
    • Transformación: Se realizan ajustes para la estandarización de montos y normalización de los beneficiarios.
    • Carga: Los datos se cargan en una base de datos destinada al análisis de impacto social.

    Fragmentos de Código Importantes¶

    # Extracción de datos
query = 'SELECT * FROM subsidios'
df_subsidios = pd.read_sql(query, engine)

# Transformación de datos
df_subsidios['monto_subsidio'] = df_subsidios['monto_subsidio'].apply(lambda x: round(x, 2))

# Carga de datos
df_subsidios.to_sql('subsidios_procesados', con=engine, if_exists='replace', index=False)

    2.4 - CrearFactFosfec¶

    Explicación del Proceso ETL¶

    El proceso de creación de FactFosfec está orientado a consolidar las contribuciones y deducciones relacionadas con el fondo de solidaridad FOSFEC, asegurando una correcta trazabilidad.

    Extracción, Transformación y Carga de Datos¶

    • Extracción: Los datos se extraen de bases internas y externas que aportan información sobre contribuciones.
    • Transformación: Se realizan verificaciones cruzadas para evitar duplicados y se calculan las deducciones necesarias.
    • Carga: Los datos finales se cargan en una tabla específica para su análisis financiero.

    Fragmentos de Código Importantes¶

    # Extracción de datos
query = 'SELECT * FROM fosfec_contribuciones'
df_fosfec = pd.read_sql(query, engine)

# Transformación de datos
df_fosfec.drop_duplicates(subset='id', keep='first', inplace=True)

# Carga de datos
df_fosfec.to_sql('fosfec_final', con=engine, if_exists='replace', index=False)
