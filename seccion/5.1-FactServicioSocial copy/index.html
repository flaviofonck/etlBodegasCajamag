    Validador de campos repetidos¶

Los duplicados se almacenan utilizando la función StoreDuplicated, y luego se eliminan los duplicados de cada tabla con drop_duplicates(). El proceso se registra en el log para cada tabla, junto con el tiempo total del validador de duplicados.

    Transformación y limpieza de texto¶

    Se realiza una limpieza de las columnas de texto en cada tabla de df_structure. Las columnas de texto se convierten a mayúsculas (UPPER), se eliminan los espacios en blanco al inicio y al final, y se reemplazan los valores "NAN" y "NONE" por valores nulos (NaN). El tiempo total del proceso de limpieza se registra en el log.

    #Transformación 2
limpieza_time = time.time()
for ky in list(df_structure.keys()):
    #print(ky)
    #Pasar las columnas tipo texto a UPPER
    df_structure[ky] = df_structure[ky].apply( lambda x: x.astype(str).str.upper())
    #En las columnas tipo texto, eliminar espacios al comienzo y al final
    df_structure[ky] = df_structure[ky].apply( lambda x: x.astype(str).str.strip())
    #Reemplazar los "NAN" o "NONE" por NaN
    df_structure[ky] = df_structure[ky].replace('NAN', np.nan)
    df_structure[ky] = df_structure[ky].replace('NONE', np.nan)

    Unión de DataFrames con registro en DWH¶

    Este código identifica y elimina columnas con más del 95% de datos vacíos en tres DataFrames: valoracion, registros y acudientes. Posteriormente, realiza una union entre estas tablas limpias, comenzando con la fusión de acudientes con registros, seguida por la unión con la tabla valoracion. El DataFrame resultante, final_merged_limpio, se guarda en la tabla BD_FactServicioSocial dentro del DWH utilizando la función guardar_en_dwh, reemplazando los datos existentes si es necesario.

    # Asumiendo que valoracion_df, registros_df y acudientes_df están en df_structure
valoracion_df = df_structure['valoracion']
registros_df = df_structure['registros']
acudientes_df = df_structure['acudientes']

# Función para calcular las columnas con más del threshold% de datos vacíos
def columnas_con_muchos_nulos(df, threshold=0.95):
    porcentaje_nulos = df.isnull().mean()
    columnas_excluir = porcentaje_nulos[porcentaje_nulos > threshold].index
    return columnas_excluir

# Verificar qué columnas tienen más del 95% de datos vacíos y eliminarlas
threshold = 0.95
columnas_acudientes_excluir = columnas_con_muchos_nulos(acudientes_df, threshold)
columnas_registros_excluir = columnas_con_muchos_nulos(registros_df, threshold)
columnas_valoracion_excluir = columnas_con_muchos_nulos(valoracion_df, threshold)

# Excluir las columnas identificadas en cada DataFrame
acudientes_df_limpio = acudientes_df.drop(columns=columnas_acudientes_excluir)
registros_df_limpio = registros_df.drop(columns=columnas_registros_excluir)
valoracion_df_limpio = valoracion_df.drop(columns=columnas_valoracion_excluir)

# Realizar las uniones con las tablas ya filtradas
# Paso 1: Unir la tabla de acudientes con registros
acudientes_registros_merged = pd.merge(acudientes_df_limpio, registros_df_limpio, left_on='registro_id', right_on='id', suffixes=('_acudiente', '_registro'))

# Paso 2: Unir el resultado anterior con la tabla de valoraciones
final_merged_limpio = pd.merge(acudientes_registros_merged, valoracion_df_limpio, left_on='registro_id', right_on='registro_id', suffixes=('', '_valoracion'))

guardar_en_dwh(final_merged_limpio, 'BD_FactServicioSocial', logger, multiple=False, if_exists='replace')
