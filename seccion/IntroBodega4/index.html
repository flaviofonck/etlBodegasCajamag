    Explicación del Proceso ETL - Bloque 4¶

    4.1 - FactColegio¶

    Explicación del Proceso ETL¶

    Este proceso tiene como objetivo procesar la información relacionada con los registros de colegios, garantizando que los datos se extraigan, transformen y carguen adecuadamente para facilitar el análisis de la gestión escolar. Se enfoca en estandarizar los registros de estudiantes y colegios para su posterior análisis.

    Extracción, Transformación y Carga de Datos¶

    • Extracción: Los datos se extraen de una base de datos que contiene la información de colegios y estudiantes, utilizando la librería sqlalchemy para conectarse y realizar consultas SQL que obtengan los registros pertinentes.
    • Transformación: Se utilizan pandas y numpy para realizar diversas tareas, como la limpieza de datos nulos, la normalización de los nombres de colegios, y el ajuste de formatos de fechas.
    • Carga: Los datos transformados se cargan en una base de datos destino, garantizando la integridad de los registros para futuros análisis de gestión.

    Fragmentos de Código Importantes¶

    import sqlalchemy as sa
from sqlalchemy import create_engine, MetaData, Table, insert
import pandas as pd
import numpy as np
from dateutil import parser

# Creación del motor de conexión a la base de datos
engine = create_engine('mysql+pymysql://usuario:contraseña@localhost/db_colegios')

# Extracción de datos
query = 'SELECT * FROM registros_colegios'
df_colegios = pd.read_sql(query, engine)

# Transformación de datos
df_colegios['fecha_registro'] = pd.to_datetime(df_colegios['fecha_registro'])
df_colegios['nombre_colegio'] = df_colegios['nombre_colegio'].str.upper()
df_colegios.fillna({'telefono': 'No Disponible'}, inplace=True)

# Carga de datos transformados
df_colegios.to_sql('colegios_procesados', con=engine, if_exists='replace', index=False)

    4.2 - FactAsistencia¶

    Explicación del Proceso ETL¶

    Este proceso tiene como propósito gestionar y consolidar la información de asistencia de los estudiantes, asegurando que los datos se procesen adecuadamente para análisis de asistencia y deserción escolar.

    Extracción, Transformación y Carga de Datos¶

    • Extracción: Se obtienen los datos de asistencia desde registros digitales utilizando sqlalchemy para acceder a la base de datos y extraer la información necesaria.
    • Transformación: Se utilizan pandas para normalizar los registros de asistencia, eliminar duplicados y convertir las fechas de asistencia a un formato estándar.
    • Carga: Los datos procesados se almacenan en una tabla específica para la generación de reportes de asistencia.

    Fragmentos de Código Importantes¶

    # Extracción de datos de asistencia
query = 'SELECT * FROM registros_asistencia'
df_asistencia = pd.read_sql(query, engine)

# Transformación de datos
df_asistencia['fecha_asistencia'] = pd.to_datetime(df_asistencia['fecha_asistencia'])
df_asistencia.drop_duplicates(subset=['id_estudiante', 'fecha_asistencia'], keep='first', inplace=True)

# Carga de datos transformados
df_asistencia.to_sql('asistencia_procesada', con=engine, if_exists='replace', index=False)
