    Importación de librerías y funciones¶

    Se importan las librerías necesarias, como sqlalchemy, pandas, numpy, y dateutil, junto con funciones personalizadas desde el archivo Funciones.py. Además, se ajusta el sys.path para incluir el directorio correspondiente, lo que asegura la correcta importación de las funciones.

    import sqlalchemy as sa

    Definición de columnas con sus tipos¶

    Se clasifican las columnas del diccionario df_structure en tres categorías: columnas numéricas, columnas de fechas (aquellas que comienzan con "fec") y columnas de texto. Las columnas de fechas se convierten al formato de fecha usando pd.to_datetime() con manejo de errores. Las listas de columnas se almacenan en tres diccionarios: numericColumns, datesColumns y textColumns. El tiempo total del proceso se registra en el log.

    #Definir columnas con sus tipos
transfor2_time = time.time()
numerics = ['int16', 'int32', 'int64', 'float16', 'float32', 'float64']
numericColumns = dict()
datesColumns = dict()
textColumns = dict()

for ky in list(df_structure.keys()):
    numericColumns[ky] = df_structure[ky].select_dtypes( include = numerics ).columns.tolist()
    datesColumns[ky] = [x for x in df_structure[ky].columns.tolist() if x.startswith('fec') ]
    textColumns[ky] = [x for x in df_structure[ky].columns.tolist() if x not in ( numericColumns[ky] + datesColumns[ky] ) ]+ [x for x in df_structure[ky].columns.tolist() if x.startswith('cod') ]+ [x for x in df_structure[ky].columns.tolist() if x.startswith('ced') ]+ [x for x in df_structure[ky].columns.tolist() if x.startswith('num') ]
    textColumns[ky] = list(set(textColumns[ky]))
    #Convertir columnas que tengan "fec" en el nombre en tipo fecha
    df_structure[ky][datesColumns[ky]] = df_structure[ky][datesColumns[ky]].apply( lambda x: pd.to_datetime( x , errors='coerce' ) )

logger.info(f'TRANSFORMACION 2 --- {time.time() - transfor2_time:.2f} seconds ---')
