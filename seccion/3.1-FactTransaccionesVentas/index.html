    Definición de consultas SQL¶

    Se definen dos consultas SQL en el diccionario qr_structure para extraer datos de las tablas servi20 y subvi21 de la base de datos servicios. Las consultas seleccionan varias columnas clave relacionadas con ventas y coberturas. El diccionario dim_names se utiliza para mapear cada consulta a su respectiva tabla en la base de datos de destino. El proceso de lectura de consultas se registra en el log.

    Validador de campos repetidos¶

    Se verifica la existencia de registros duplicados en todas las tablas de df_structure. Las columnas a comparar se determinan excluyendo la columna id. Los duplicados se almacenan utilizando la función StoreDuplicated, y luego se eliminan los duplicados de cada tabla con drop_duplicates(). El proceso se registra en el log para cada tabla, así como el tiempo total del validador de duplicados.

    #Validador de campos repetidos 
validador_time = time.time()
for ky in list(df_structure.keys()):   
    ColumnsToCompare = [x for x in df_structure[ky].columns.tolist() if x not in [ 'id' ] ]
    StoreDuplicated('N/A' , ColumnsToCompare, df_structure[ky], r'trazaDuplicados_' + ky )
    #Limpieza inicial quitar duplicados
    df_structure[ky] = df_structure[ky].drop_duplicates()
    logger.info('VALIDADOR TABLA: '+ ky)
logger.info(f'VALIDADOR DUPLICADOS --- {time.time() - validador_time:.2f} seconds ---')

    Transformación y limpieza de texto¶

    Se realiza una limpieza de las columnas de texto en cada tabla de df_structure. Las columnas de texto se convierten a mayúsculas (UPPER), se eliminan los espacios en blanco al inicio y al final de cada valor, y se reemplazan los valores "NAN" y "NONE" por NaN. El tiempo total del proceso de limpieza se registra en el log.

    #Transformación 2
limpieza_time = time.time()
for ky in list(df_structure.keys()):
    #print(ky)
    #Pasar las columnas tipo texto a UPPER
    df_structure[ky] = df_structure[ky].apply( lambda x: x.astype(str).str.upper())
    #En las columnas tipo texto, eliminar espacios al comienzo y al final
    df_structure[ky] = df_structure[ky].apply( lambda x: x.astype(str).str.strip())
    #Reemplazar los "NAN" o "NONE" por NaN
    df_structure[ky] = df_structure[ky].replace('NAN', np.nan)
    df_structure[ky] = df_structure[ky].replace('NONE', np.nan)
logger.info(f'LIMPIEZA --- {time.time() - limpieza_time:.2f} seconds ---')

    2024-10-13 18:42:59,215 - INFO - LIMPIEZA --- 75.40 seconds ---
