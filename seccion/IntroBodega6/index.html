    Explicación del Proceso ETL - Bloque 6¶

    6.1 - MainEncuestasV1¶

    Explicación del Proceso ETL¶

    Este proceso se encarga de gestionar la información relacionada con encuestas realizadas a diferentes grupos, con el propósito de consolidar los datos para su posterior análisis estadístico y evaluación de tendencias. A través de este proceso, se asegura la correcta extracción, transformación y carga de los datos de encuestas, garantizando la calidad de los datos y su disponibilidad para análisis posteriores.

    Extracción, Transformación y Carga de Datos¶

    • Extracción: Los datos se extraen de una base de datos de encuestas utilizando sqlalchemy para conectarse y realizar consultas SQL que permitan obtener los registros relevantes.
    • Transformación: Se utilizan pandas para realizar diversas tareas de transformación, como la limpieza de valores nulos, la estandarización de respuestas, y la conversión de formatos de fecha.
    • Carga: Los datos transformados se cargan en una base de datos destino para ser utilizados en análisis y reportes.

    Fragmentos de Código Importantes¶

    import sqlalchemy as sa
from sqlalchemy import create_engine, MetaData, Table, insert
import pandas as pd
import numpy as np
from dateutil import parser

# Creación del motor de conexión a la base de datos
engine = create_engine('mysql+pymysql://usuario:contraseña@localhost/db_encuestas')

# Extracción de datos
query = 'SELECT * FROM encuestas_v1'
df_encuestas = pd.read_sql(query, engine)

# Transformación de datos
df_encuestas['fecha_encuesta'] = pd.to_datetime(df_encuestas['fecha_encuesta'])
df_encuestas.fillna({'respuesta': 'No Respuesta'}, inplace=True)
df_encuestas['respuesta'] = df_encuestas['respuesta'].str.upper()

# Carga de datos transformados
df_encuestas.to_sql('encuestas_procesadas', con=engine, if_exists='replace', index=False)

    6.2 - ResultadosEncuestas¶

    Explicación del Proceso ETL¶

    Este proceso tiene como objetivo consolidar los resultados de las encuestas para generar reportes y análisis que ayuden a identificar tendencias y preferencias de los diferentes grupos encuestados. La finalidad es asegurar que los datos sean consistentes y útiles para la toma de decisiones.

    Extracción, Transformación y Carga de Datos¶

    • Extracción: Se extraen los datos ya procesados desde la tabla de encuestas para generar un resumen de los resultados.
    • Transformación: Se agrupan las respuestas por grupo objetivo y se calculan métricas como la cantidad de respuestas y la distribución porcentual de cada tipo de respuesta.
    • Carga: Los datos consolidados se cargan en una nueva tabla para reportes de resultados y análisis de tendencias.

    Fragmentos de Código Importantes¶

    # Extracción de datos procesados
query = 'SELECT * FROM encuestas_procesadas'
df_resultados = pd.read_sql(query, engine)

# Transformación de datos
df_agrupado = df_resultados.groupby(['grupo_objetivo', 'respuesta']).size().reset_index(name='cantidad_respuestas')
df_agrupado['porcentaje'] = (df_agrupado['cantidad_respuestas'] / df_agrupado.groupby('grupo_objetivo')['cantidad_respuestas'].transform('sum')) * 100

# Carga de resultados
resultados_tabla = 'resultados_encuestas'
df_agrupado.to_sql(resultados_tabla, con=engine, if_exists='replace', index=False)
