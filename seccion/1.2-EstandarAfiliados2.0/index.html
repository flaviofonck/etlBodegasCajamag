    Definición de tipos de columnas¶

    Aquí se identifican las columnas del dataframe dfTotal según su tipo de dato: numéricas, de texto y de fechas. Se crean listas con los nombres de las columnas correspondientes a cada tipo de dato, lo cual será útil para las operaciones posteriores.

    #Definir columnas con sus tipos

dates = ['datetime64[ns]']
numericColumns = dfTotal.select_dtypes(include=numerics).columns.tolist()
datesColumns = dfTotal.select_dtypes(include=dates).columns.tolist()
textColumns = [x for x in dfTotal.columns.tolist() if x not in (numericColumns + datesColumns ) ]
ColumnsToCompare = [x for x in dfTotal.columns.tolist() if x not in [ 'ID' , 'FECHA_ESTADO' ,'xml4c086', 'xml4c085', 'SUBSI20', 'xml4c087' ] ]

    Segunda fase de transformación: agregación por ID¶

    Esta celda agrupa los datos por la columna ID, que representa la clave única de los registros, y realiza una agregación de los valores usando la función unique. Esta agregación ayuda a identificar los valores únicos en cada columna por cada ID, y se registra el tiempo tomado para esta transformación.

    tratamiento2_time = time.time()
#Cambian None por NaN
dfTotal[textColumns] = dfTotal[textColumns].where(pd.notna(dfTotal), np.nan)

#Pasar las columnas tipo texto a UPPER
dfTotal[textColumns] = dfTotal[textColumns].apply( lambda x: x.astype(str).str.upper())

#En las columnas tipo texto, eliminar espacios al comienzo y al final
dfTotal[textColumns] = dfTotal[textColumns].apply( lambda x: x.astype(str).str.strip())

#Reemplazar los "NAN" por NaN
dfTotal[textColumns] = dfTotal[textColumns].replace('NAN', np.nan)

#ValIDador de campos repetIDos
ValIDador = dfTotal.groupby('ID')[ ColumnsToCompare ].nunique(dropna=False).reset_index()
ValIDador['suma']  = ValIDador.sum(axis=1, numeric_only=True)

#Tablas de información para guia transformación
IDProblemas = ValIDador[ValIDador['suma'] > len(ColumnsToCompare) ]['ID'].tolist()
dfRepetIDos = dfTotal[ dfTotal['ID'].isin(IDProblemas)  ].groupby('ID')[ ColumnsToCompare ].agg(['unique']).reset_index()
dfRepetIDos.columns = dfRepetIDos.columns.droplevel(1)
trazaDf = pd.merge(dfRepetIDos,dfOrigen,on = 'ID' )
